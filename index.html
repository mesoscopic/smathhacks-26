About

This project was created for SMATH HACKS 2026!

What is it?

This is a project that allows you to listen to procedurally generated music that uses oceanographic data. It fetches data continuously to stay up to data and moves your location according to ocean currents to keep the music fresh.

What data is used?

We are using NOAA buoy data and NOAA CoastWatch data to get us real-time wave information like wave height and wave direction, water temperatures, wind speeds and wind directions.

What tools did we use?

    We used:
  1. HTML, CSS, and Typescript: to build the website.
  2. Tone.js to synthesize the music.
  3. Python to build the web server and fetch data through the APIs.
